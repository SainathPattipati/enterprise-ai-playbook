# Contributing to Enterprise AI Playbook

We welcome contributions from AI practitioners. This is a living document that improves with community input.

## How to Contribute

### Reporting Issues
Found an error or outdated information? Open an issue with:
- What chapter/section
- What's wrong
- What should it be instead

### Suggesting Improvements
- Open an issue with your suggestion
- Provide context (what use case, what company size)
- Examples are helpful

### Adding Content
Have a pattern or lesson learned to share?
- Fork the repository
- Add your content in appropriate section
- Submit pull request with clear title
- Include your experience/context

### Content Guidelines
- Write from practitioner perspective
- Include real numbers/examples when possible
- Be honest about tradeoffs
- Avoid vendor bias
- Use clear, accessible language

## Content Structure

Each chapter should include:
- **Overview** - What this is about
- **The Problem** - Why this matters
- **Approach** - How to do it
- **Pitfalls** - What goes wrong
- **Success Metrics** - How to measure
- **Getting Started** - First steps

## Attribution

Contributors will be credited:
- Your name in the chapter
- Your GitHub username
- Brief bio (if desired)

## Questions?

Open an issue or discussion in the repository.
